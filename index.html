망라 네트워크 스캐너

포괄적인 네트워크 탐색 및 보안 분석 도구

개발자: 김지훈

주요 기능

다중 포트 스캔

동시 다중 포트 스캐닝으로 효율적인 네트워크 매핑

네트워크 발견

활성 호스트 자동 탐색 및 식별

서비스 인식

열린 포트의 서비스 유형 자동 인식

크로스 플랫폼

Windows, Linux, macOS에서 동작 보장

설치 방법

# 저장소 복제
git clone https://github.com/joseon-dev/망라-네트워크-스캐너.git

# 디렉토리 이동
cd 망라-네트워크-스캐너

# 실행 권한 부여
chmod +x network_scanner_plus.py

사용 예제

# 기본 포트 스캔
python3 network_scanner_plus.py

# 특정 포트 지정 스캔
python3 network_scanner_plus.py --ports 22,80,443 --target 192.168.1.1

# 네트워크 대역 스캔
python3 network_scanner_plus.py --network 10.0.0.0/24

호환성 정보

  • Python 3.6 이상 필요
  • 표준 라이브러리만 사용 - 추가 의존성 없음
  • 조선 네트워크 환경에서 테스트 완료
  • 코리올링크 3G/4G 네트워크 호환
